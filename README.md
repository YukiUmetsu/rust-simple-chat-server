# rust-simple-chat-server
simple chat server with rust<br>

# start the server
1. `cd chat_server`<br>
2. `cargo run`<br>
<br>
# start a client
1. `cd chat_client`<br>
2. `cargo run`<br>
* you can run multiple clients to chat<br>
<br>
# how to use the chat
1. type your name<br>
2. enjoy the chat!<br>
